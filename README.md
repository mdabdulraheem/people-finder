# People Finder
## A simple people finder search app built using:
- NextJs
- Typescript
- Tailwind CSS
- ShadCN UI
- jsonplaceholder API (https://jsonplaceholder.typicode.com/)

## Getting Started

- First Clone the repo
- cd people-finder
- Run the development server:
```bash
npm install
```
```bash
 npm run dev
```


You can use yarn or pnpm or bun instead of npm.
